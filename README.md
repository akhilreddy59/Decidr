## 🔭 Overview
**Decidr** (Project AIML5) is a generative intelligence platform designed to navigate the "Fundamental Mismatch" between deterministic software requirements and non-deterministic AI behavior. 

Unlike traditional PRD tools, Decidr utilizes a **3D Spatial Reasoning Engine** and **Chain-of-Thought (CoT) logic** to analyze product goals, detect constraint contradictions, and generate explainable architectural paths.

## 🧠 Core Intelligence Features
### 1. The Conflict Detector (Red-Alert Logic)
An autonomous check that flags "Elasticity Threshold" violations. If you input "High Security" with a "Zero Budget," the engine triggers a visceral conflict alert before proceeding.

### 2. Trade-off Matrix
A multi-dimensional scoring system that evaluates competing paths based on:
- **Feasibility (Technical Depth)**
- **Risk (Probabilistic Failure Rate)**
- **Velocity (Speed to Market)**

### 3. The Audit Trail
Full transparency into the AI's internal monologue. View the step-by-step reasoning chain that led to a specific recommendation.

## 🛠 Technical Stack
- **Frontend:** Next.js 15 (App Router), Tailwind CSS v4
- **3D Engine:** React Three Fiber (Three.js), GSAP (Scrub-Animations)
- **Motion:** Framer Motion, Lenis (Smooth Scroll)
- **Intelligence:** Claude 3.5 Sonnet (via Groq/Anthropic), FastAPI
- **State:** Zustand (Physics-based state management)
